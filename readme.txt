Flashcards App for Learning Spanish
A web-based flashcards app built with TypeScript, React, and Vite, designed to help users learn Spanish vocabulary. The app features card flipping, quiz modes, user progress tracking, and a statistics page.
Features
* Flashcards: Flip cards to learn Spanish vocabulary and see the English translation.
* Quiz Mode: Randomized multiple-choice or fill-in-the-blank questions based on flashcards.
* User Progress: Tracks correct and incorrect answers, storing data locally for progress between sessions.
* Statistics Page: Displays overall progress, including total cards studied, correct vs. incorrect answers, and review suggestions.
* Responsive Design: The app is fully responsive, ensuring a seamless experience on both desktop and mobile devices.
Technologies Used
* Frontend:
o TypeScript
o React
o Vite
o CSS for styling (or CSS-in-JS libraries like styled-components if preferred)
* Data Storage: Local storage to track user progress
Installation
To get started with the project, follow these steps:
1. Clone the repository:
2. git clone https://github.com/your-username/flashcards-app.git
3. cd flashcards-app
4. Install dependencies:
5. npm install
6. Run the development server:
7. npm run dev
8. Open http://localhost:3000 in your browser to see the app in action.
Features
Flashcards
* Display Spanish words with the ability to flip the card to reveal the English translation.
* Track whether the user got the card right or wrong with feedback buttons ("Correct" and "Incorrect").
Quiz/Test Mode
* Multiple Choice: Randomized quiz with 3-4 options, one correct and the rest incorrect.
* Fill-in-the-Blank: A blank space for the user to fill with the correct English translation.
Statistics Page
* Track the number of cards studied.
* Show the percentage of correct vs. incorrect answers.
* Keep track of incorrect cards for future review.
Roadmap
Future improvements and features might include:
* Adding more languages for flashcards.
* Syncing user progress with a backend server.
* Implementing spaced repetition algorithms for smarter learning.
